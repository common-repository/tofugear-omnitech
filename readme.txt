A WordPress plugin to integrate Tofugear Omnitech; A fully customized omni-channel retailing platform.

== Installation ==

1. Upload `tofugear-omnitech` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
